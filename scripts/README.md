# Scripts Directory

This directory contains helper scripts for the Compound Engineering workflow.

> **Attribution**: These automation scripts implement concepts from the [Compound Engineering Plugin](https://github.com/EveryInc/compound-engineering-plugin) by EveryInc.

## Available Scripts

### Git Worktree Management

- `git-worktree.ps1` - PowerShell script for managing git worktrees
- `git-worktree.sh` - Bash script for managing git worktrees (Unix/Linux/macOS)

## Usage

See individual script files for detailed usage instructions.

## Platform Support

- Windows: Use `.ps1` PowerShell scripts
- Unix/Linux/macOS: Use `.sh` bash scripts
